Backup Data Sistem

Fitur ini memungkinkan Anda untuk mengunduh seluruh data aplikasi saat ini (termasuk profil sekolah, kunci jawaban, dan data jawaban siswa) ke dalam sebuah file berekstensi .json. Gunakan fitur ini untuk mencadangkan data Anda agar tidak hilang atau jika Anda ingin memindahkan data ke perangkat/laptop lain.

Restore (Pulihkan) Data

Perhatian!

Melakukan proses restore (pemulihan) akan menimpa dan menghapus seluruh data yang sedang aktif di aplikasi saat ini dengan data yang ada di dalam file backup Anda.

Silakan pilih file .json yang sebelumnya telah Anda unduh dari fitur backup di atas.
